IT Tech Adventure Game

Game Addendum

HTML Elements:

  • <header>: Contains the title of the game.
  • <main>: Main content area where the story and choices are displayed.
  • <div id="story">: Contains the current stage's text and image.
  • <div id="choices">: Contains buttons for player choices at each stage.
  • <footer>: Displays the last modified date of the page.
  • <button>: Buttons used for navigating through the choices and restarting the game.
  • <script>: Links the external JavaScript file responsible for the game logic.

JavaScript Functions:

  • startGame(): Initializes the game and sets the starting stage to 'start'.
  • updatePage(): Updates the HTML page with the text, image, and choices based on the current stage.
  • document.addEventListener('DOMContentLoaded', startGame): Ensures the game starts once the DOM content is fully loaded.
  • document.getElementById('lastModified').textContent = document.lastModified: Updates the footer with the last modified date.

CSS Elements:

  • body: Defines the overall appearance and layout of the page.
  • #story: Sets the text and image display area.
  • #choices: Aligns the choice buttons.
  • .choiceButton: Styles the choice buttons with padding, background color, and hover effects.
  • footer: Positions and styles the footer with the last modified date.

Image Assets:

  • "computer_setup.png", "coding_bootcamp.png", "debugging.png", etc.: Represent the images displayed for each stage of the game. These should be present in the same folder as the HTML file or adjust the paths as needed.

Event Handling:

  • Buttons are created dynamically for each choice and linked to the respective consequence stage.
  • When a button is clicked, the currentStage variable is updated to the choice's consequence and the page is re-rendered.

Restart Mechanism:

  • A "Restart" button is shown when there are no further choices in the current stage. Clicking this resets the game to the start stage.

This game involves navigating through different IT tech scenarios based on the player's choices, with multiple possible endings.
